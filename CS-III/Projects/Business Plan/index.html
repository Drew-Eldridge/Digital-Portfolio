ISP Proposal

IDENTITY

A computer that will shut off if someone else besides the user looks at it, or if the user looks away.

PROBLEM

As technology has progressed, the ability to remain anonymous has diminished along with it. While digital security is often the side of privacy that is most scrutinized and focused on, physical security is overlooked. This is where my product comes in. We have all experienced a little brother, a pesky classmate, an annoying stranger, a random onlooker, or just anyone who may want to peer on our screen when we don't want them too. This is the problem I am trying to solve.

SOLUTION

My solution to this problem is to create a camera that can attach to a computer and make it shut off with a moments notice whenever someone else besides the user looks at the computer, or if the user looks away for too long.

TARGET MARKET

My target market are people at work or school who are doing stuff that they should not be doing, or people in any setting that just want to be more safe with their technology usage.

COMPETITION

There is virtually no competition for something as extreme as this, though there is lots of facial recognition security features which are being implemented such as face id on the iPhone, that could be considered competition.

REVENUE STREAMS

The revenue stream is relatively straight forward, I can sell a camera which can plug into a computer and connect at the top of it which will run my code to make their computer secure. Also I can sell a program which people with laptops with cameras can download which will allow them to have the same security features while still maintaing their streamline laptop.

MARKETING STREAMS

I can market this product through sites like tiktok since I feel that a lot of users on those sort of platforms can relate to the problem that this product is attempting to solve. I could also try to get deals with products such as VPNs to see if they could advertise my product on their website since while we are not competitors we are in related fields and could benefit off of working together.

EXPENSES

My expenses would mostly be the cameras if I choose to go with the cameras and not just with code that would run off of any camera. Besides that there is really no other expense since it is all software based.

TEAM AND KEY ROLES

There can be a head of the project (AKA ME), as well as a head software engineer, then just software engineers that help with the code.
